# AI-Assisted Face Detector Web Application

This project was built for the Advanced Programming course at the University of Pisa (UniPi) for the academic year 2025/2026.

The application is a browser-based real-time face detector that uses the user’s webcam to detect faces and display bounding boxes in real time.

## Features

* Real-time face detection
* Live webcam input
* Bounding box overlay
* FPS counter
* Start/Stop detection toggle
* Fully client-side processing

## Technologies Used

* HTML
* CSS
* JavaScript
* MediaPipe Tasks-Vision

## How to Run

Open the live [demo link](https://abderrahmane-salmi.github.io/ai-face-detector-webapp/), allow camera access, and the application will start detecting faces in real time.

You can also run the project locally using VS Code and the Live Server extension.

## Privacy

All face detection is performed locally in the browser. No images, videos, or personal data are uploaded or stored on a server.
